Responsive Design Features

  • Flexible Layouts: Sử dụng CSS Grid và Flexbox
  • Media Queries: 6 breakpoints chính (375px, 576px, 768px, 992px, 1200px, 1400px)
  • Touch Optimization: Button size ≥ 44x44px cho mobile
  • Performance: Lazy loading images, optimized animations
  • Accessibility: Semantic HTML, ARIA labels, keyboard navigation
  • Font Scaling: Responsive typography với rem/em units
  • Spacing: Adaptive padding/margin cho các screen sizes
